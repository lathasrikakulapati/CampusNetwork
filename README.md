# CampusNetwork
Campus Network is a web based platform developed to help students prepare for campus placements and stay updated with recruitment activities. The main goal of this project is to provide a centralized system where students can access placement information, practice interview questions, attempt quizzes, and track their preparation progress.

What This Project Does

The application allows students to create an account and log in securely. After logging in, students can access a personalized dashboard where they can explore various placement related resources.

It provides information about upcoming placement drives and events so that students can stay informed and plan accordingly. Students can also practice mock interview questions and review commonly asked technical and HR interview questions to improve their confidence and skills.

The platform includes a quiz system where students can attempt tests and track their scores. This helps them evaluate their knowledge and identify areas that need improvement.

There is also a feedback feature that allows students to share their suggestions or concerns. This helps improve the overall system experience.

For administrators, the project provides a separate dashboard to manage users, placement events, interview questions, quizzes, FAQs, and feedback. Admins can add, update, or delete content, ensuring that the information available to students is accurate and up to date.

Overall Purpose

The Campus Network project acts as a digital placement preparation assistant. It connects students with placement resources, organizes preparation materials in one place, and simplifies the management of placement related activities within a campus environment.
